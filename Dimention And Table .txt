            			 position
===================================================
The position property specifies the type of positioning method used for an element.


There are five different position values:

static :- An element with position: static; is not positioned in any special way; it is always positioned according to the 

relative :- An element with position: relative; is positioned relative to its normal position

fixed :- An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same 
             place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.

absolute :-  An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

sticky :-  An element with position: sticky; is positioned based on the user's scroll position.
