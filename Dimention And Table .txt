
Dimension
------------------------
height and width
height :- top to bottom
width :- left to right

Values:- auto ,length, %, initial, inheit
auto:- this is default. The browser calculates the 
       height and width
	   
length:- Defines the height/width in px, cm etc.

% :- Defines the height/width in percent of 
the containing block

initial:- Sets the height/width to 
           its default value
		   
inherit:- The height/width will be inherited 
from its parent value
     Table border
=========================
border :- the set table, table-head, table-data border using table,th,td

border-collapse:- property sets whether the table borders should be collapsed into 
                    a single border
					
					
Table Size:- use to width on table set,table-head set height

Alignment:- property sets the horizontal alignment (like left, right, or center) 
             of the content in <th> or <td>	 


            Table Style	
===================================

Table Padding:- To control the space between the border and the content in a table, use the
               padding property on <td> and <th> elements
				
Horizontal Dividers:- Add the border-bottom property to <th> and <td> for horizontal dividers				

Hoverable Table:-  use to :hover selector on <tr>to highlight table rows on mouse over
	
Table Color:-The example below specifies the background color and text color of <th> elements

	   
	   
	 		Responsive Table
===============================
overflow:- Add a container element (like <div>) with overflow-x:auto around the <table> element 
		   to make it responsive
